# Medicare Healthcare Website

## Overview

Medicare is a responsive healthcare company website developed using HTML, CSS, and JavaScript. The project demonstrates a professional landing page design suitable for hospitals, clinics, healthcare providers, and medical organizations.

## Features

* Responsive Design
* Modern User Interface
* Fixed Navigation Bar
* Hero Section
* About Us Section
* Services Section
* Healthcare Statistics
* Contact Section
* Interactive JavaScript Features
* Smooth Scrolling Navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
